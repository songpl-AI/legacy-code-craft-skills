# Step 5-6: 自检清单 + 输出修改说明

## Step 5 — 自检清单

修改完成后逐一核对：

- [ ] 是否优先采用了新增方式，修改是否必要且已在风险报告说明
- [ ] 关键路径是否都有 g_LogError 日志
- [ ] HTTP 请求是否有 fail 回调
- [ ] Watch 参数是否用 args[0]
- [ ] 字符串连接是否用 `..`
- [ ] 数值是否 tostring()
- [ ] Exit() 是否清理了新增资源
- [ ] nil 安全检查是否到位
