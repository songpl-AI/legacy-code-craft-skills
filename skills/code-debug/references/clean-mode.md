# 模式2：清除调试日志

## Step 1 — 读取脚本
完整读取目标脚本

## Step 2 — 查找所有 [DEBUG] 日志
搜索所有包含 `[DEBUG]` 的 `g_LogError` 行

## Step 3 — 逐一删除
将每一行含 `g_LogError("[DEBUG]` 的代码行删除（包括行首的空白缩进）

**注意事项**：
- 如果 `[DEBUG]` 日志在 if 块中且 if 块只有这一条语句，需要连 if 一起删除
- 删除后检查是否产生空行堆积，保持代码整洁

## Step 4 — 验证清除完成

清除后**必须**搜索确认无残留：
- 搜索 `g_LogError("[DEBUG]` 确认剩余 0 条
- 如果仍有残留，继续清除直到全部移除

## Step 5 — 输出清除报告

```
## 调试日志清除报告

**脚本**: 文件名
**清除数量**: N 条 [DEBUG] 日志已移除
**剩余 [DEBUG]**: 0 条（已验证全部清除）
```
